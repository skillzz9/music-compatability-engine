# FlowState: AI-Integrated Audio Looping Platform

FlowState is a high-performance audio engine designed to bridge the gap between machine learning and live music performance. This project serves as a technical thesis and personal endeavor to automate complex DJ workflows through deep learning.

## Core Features

* **Siamese Neural Network (SNN):** Utilizes a custom-trained SNN to mathematically determine compatibility between diverse audio tracks.
* **AI DJ Engine:** Leverages machine learning to mix content harmonically, automating the transition process while maintaining musical integrity.
* **Dual-Mode Time-Stretching:** Employs advanced signal processing for phase-alignment, ensuring seamless synchronization during real-time playback.
* **Low-Latency Stack:** Architected for professional-grade performance to ensure a reliable and high-quality user experience during live use.
* **ML Emotion Mapping:** Integrates data-driven emotion analysis to suggest track improvements and analyze listener resonance.


## Project Goals

This project is currently in active development for 2026. 

1. **Neural Training:** Siamese Neural Network training and validation for high-accuracy track matching.
2. **Real-Time Integration:** Implementation of the low-latency audio stack for zero-lag performance.
3. **Harmonic Mixing:** Deployment of the AI DJ feature to handle complex transitions.
4. **Sentiment Analysis:** Refinement of emotion mapping algorithms to track audience engagement.
